# 모노폴리 랜덤 룰렛

확률과 선택지를 넣어 결과를 산출하는 간단한 시스템 룰렛입니다.

# 링크

[바로가기](https://gaseu-rangers.github.io/random_roulette/)

## 사용 방법
1. 이름과 확률을 입력합니다.
2. 필요 시 '선택지 추가' 버튼을 통해 선택지를 추가합니다.
3. 룰렛 돌리기 버튼을 누릅니다.

- 기입된 확률(%) 입력값의 총합이 정확히 100일 때만 실행됩니다.
- 입력값은 finite number만 허용됩니다.
- NaN / ±Infinity / string / empty value / negative number는 허용되지 않습니다.
- 0 이상의 int / float 값만 입력 가능합니다.
